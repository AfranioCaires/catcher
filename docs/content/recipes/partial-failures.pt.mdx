---
title: Tratando Falhas Parciais
description: Processando listas de itens onde falhas individuais não devem interromper todo o processo.
---

## O Problema

Ao processar um Batch de itens (como enviar múltiplos e-mails ou atualizar muitos registros), uma única falha muitas vezes não deve derrubar todo o trabalho. Você precisa saber quais itens tiveram sucesso e quais falharam para gerar um relatório.

## A Solução

Combine o processamento em Batch com o particionamento para separar sucessos de falhas em duas listas distintas.

<CodeBlock
    filename="bulk-email.ts"
    language="typescript"
    code={`async function processJob(items: string[]) {
    const results = await catchErrorAll(
        items.map(id => catchError(sendEmail(id)))
    );

    const { ok: sent, err: failed } = partition(results);

    console.log(\`Successfully sent: \${sent.length}\`);
    console.log(\`Failed: \${failed.length}\`);

    return { sent, failed };
}`}
/>
