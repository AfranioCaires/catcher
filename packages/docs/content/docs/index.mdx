# Getting Started

The **Catcher** provides a robust way to handle errors using the Result pattern, eliminating the need for deeply nested control flow blocks.

<Steps>
<Step>
### Install the package

Use your preferred package manager to add the library to your project.

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
    ```bash
    npm install @catcherjs/core
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @catcherjs/core
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @catcherjs/core
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @catcherjs/core
    ```
  </Tab>
</Tabs>
</Step>

<Step>
### Basic Implementation

Import the function and wrap your promise.

```typescript
import { catchError } from '@catcherjs/core'

async function fetchUser(userId: string) {
  const [error, data] = await catchError(fetch(`https://api.example.com/users/${userId}`))

  if (error) {
    throw new Error('Network failure')
  }

  return data.json()
}
```

</Step>
</Steps>

<Callout title="TypeScript Integration" type="info">
  The library is fully typed and infers the return types automatically based on the wrapped
  function.
</Callout>
