# Processamento em Lote

Se você precisa executar uma lista de operações e coletar todos os seus resultados—sem deixar que uma única falha interrompa todo o lote—você pode usar os utilitários de lote.

<Tabs items={['Lote Assíncrono', 'Lote Síncrono']}>
  <Tab value="Lote Assíncrono">
    Use `batchCatch` para um array de Promises ou funções assíncronas. Ele aguarda a conclusão de todos e retorna um array de objetos Result.

    ```typescript
    import { batchCatch } from 'catch-error-lib';

    async function processTasks() {
      const tasks = [
        () => Promise.resolve('Task 1 complete'),
        () => Promise.reject(new Error('Task 2 failed')),
        () => Promise.resolve('Task 3 complete')
      ];

      const results = await batchCatch(tasks);

      results.forEach(result => {
        if (result.isOk()) {
          console.log(result.data);
        } else {
          console.error(result.error.message);
        }
      });
    }
    ```
  </Tab>
  <Tab value="Lote Síncrono">
    Use `batchCatchSync` para um array de funções síncronas. Ele as executa sequencialmente e retorna um array de objetos Result.

    ```typescript
    import { batchCatchSync } from 'catch-error-lib';

    function parseMultiple() {
      const inputs = [
        () => JSON.parse('{"a": 1}'),
        () => JSON.parse('invalid'),
        () => JSON.parse('{"c": 3}')
      ];

      const results = batchCatchSync(inputs);

      const successfulParses = results
        .filter(r => r.isOk())
        .map(r => r.data);

      console.log(successfulParses);
    }
    ```
  </Tab>
</Tabs>
