Catcher é uma biblioteca moderna e leve de tratamento de erros para TypeScript que traz o padrão Result para sua base de código. Ela foi projetada para tornar o gerenciamento de erros padronizado, seguro em relação aos tipos e fluido, ajudando você a evitar o "try-catch hell" enquanto mantém um fluxo de controle claro.

Fortemente inspirada nas arquiteturas de tratamento de erros de linguagens como Go e Rust, a Catcher oferece uma abordagem híbrida que se encaixa perfeitamente no ecossistema TypeScript.

A biblioteca é completamente independente, com zero dependências, garantindo impacto mínimo no tamanho do seu bundle e na complexidade do projeto.
