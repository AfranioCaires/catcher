# Primeiros Passos

O **catch-error-lib** fornece uma maneira robusta de lidar com erros usando o padrão Result, eliminando a necessidade de blocos de controle profundamente aninhados.

<Steps>
<Step>
### Instale o pacote

Use o gerenciador de pacotes de sua preferência para adicionar a biblioteca ao seu projeto.

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
    ```bash
    npm install catch-error-lib
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add catch-error-lib
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add catch-error-lib
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add catch-error-lib
    ```
  </Tab>
</Tabs>
</Step>

<Step>
### Implementação Básica

Importe a função e envolva sua promise.

```typescript
import { catchError } from 'catch-error-lib';

async function fetchUser(userId: string) {
  const [error, data] = await catchError(fetch(`https://api.example.com/users/${userId}`));

  if (error) {
    throw new Error('Network failure');
  }

  return data.json();
}
```
</Step>
</Steps>

<Callout title="Integração com TypeScript" type="info">
A biblioteca é totalmente tipada e infere os tipos de retorno automaticamente com base na função fornecida.
</Callout>
